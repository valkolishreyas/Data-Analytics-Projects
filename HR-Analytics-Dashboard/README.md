
# HR Analytics Dashboard

## Objective
Analyze employee attrition patterns and workforce metrics using Power BI.

## Tools Used
- Power BI
- Excel/CSV
- DAX

## KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Tenure

## Key Insights
- Research & Development shows highest attrition.
- Employees aged 25–35 have the highest turnover.
- Overtime employees are more likely to leave.
- Laboratory Technicians and Sales Executives show the highest attrition.
